# 🌐 Restaurant Website & Weather Dashboard

This repository contains two web development projects developed as part of the **Pinnacle Labs 2026 Internship Program**: a **Restaurant Website** and a **Weather Dashboard**. These projects demonstrate responsive web design, API integration, and interactive user interfaces, providing practical experience in building modern web applications. :contentReference[oaicite:0]{index=0}

## 📌 Projects

### 🍽️ Restaurant Website
A responsive restaurant website featuring an attractive homepage, menu section, about page, contact information, and table reservation interface to enhance the customer dining experience.

### 🌦️ Weather Dashboard
A web application that displays real-time weather information for any city using a weather API, including temperature, humidity, wind speed, and weather conditions.

## 🚀 Features

- Responsive and modern user interface
- Real-time weather data through API integration
- Interactive navigation and layouts
- Clean and organized codebase
- Mobile-friendly design
- Easy to customize and extend

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Weather API
