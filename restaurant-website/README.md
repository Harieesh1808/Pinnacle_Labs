# 🍽️ Restaurant Website

A modern, responsive restaurant website developed as part of the **Pinnacle Labs Web Development Internship**. The project provides an engaging online presence for a restaurant, allowing visitors to explore the menu, learn about the restaurant, make table reservations, discover upcoming events, and get in touch through a clean and intuitive interface.

---

## 📖 Overview

The Restaurant Website is designed to deliver an excellent user experience by combining an attractive layout with easy navigation. It serves as a complete digital platform for customers to browse restaurant offerings, reserve tables, and explore special events.

The project emphasizes responsive design, organized project structure, and reusable front-end components using HTML, CSS, and JavaScript.
