# 🌦️ Dynamic Weather Dashboard

A modern and responsive **Weather Dashboard** built using **HTML, CSS, and JavaScript**. The application allows users to search for weather information of any city, view a 5-day weather forecast, switch between temperature units, and fetch weather data based on the user's current location.

---

## 📁 Project Structure

```
Dynamic-Weather-Dashboard/
│
├── index.html      # Main HTML file
├── styles.css      # Styling for the application
├── app.js          # JavaScript functionality
└── README.md       # Project documentation
```

---

## 🚀 Features

* 🔍 Search weather by city name
* 📍 Get weather using current location
* 🌡️ Toggle between Celsius (°C) and Fahrenheit (°F)
* 🌤️ Display current weather conditions
* 📅 View 5-day weather forecast
* 💧 Shows humidity, pressure, wind speed, and feels-like temperature
* 🎨 Modern responsive user interface
* ⏳ Loading animation while fetching weather data
* ⚠️ Error handling for invalid city names

## ⚙️ Installation

1. Download or clone the project.

```bash
git clone https://github.com/your-username/dynamic-weather-dashboard.git
```

2. Open the project folder.


3. Open **index.html** in your browser.

---

## 📖 How to Use

1. Open the application.
2. Enter a city name in the search box.
3. Select a city or press **Enter**.
4. View:

   * Current temperature
   * Weather description
   * Humidity
   * Wind speed
   * Pressure
   * Feels-like temperature
5. Click the **Location** button to get weather for your current location.
6. Click the **°C / °F** button to switch temperature units.
